# Conifer
A game engine I am working on: still in its very early stages. So far there is a shader reader and I am working on transformations and the camera. The source code can be found in the master branch.
## What its made of
Conifer is an engine which uses C++ as its main programming language.
It uses OpenGL to render and uses GLM for linear transformations in it.
I plan to have:
  1. Qt for application GUI
  2. Conan for package manager
  3. CMake for build system
